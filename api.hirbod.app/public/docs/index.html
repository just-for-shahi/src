Authentication ¶

The Authentication methods related to sign, verify, otp, captain and revoke methods.

Sign ¶

Sign user that update or create user account.

POST https://api.hirbod.ac/rest/v1/auth/sign

Headers
Content-Type: application/json
Body
{
  "mobile": "9121234567",
  "country": "+98"
}
Responses200
Headers
Content-Type: application/json
Body
{
  "meta": {
    "code": 200,
    "message": "User Signed."
  },
  "data": null
}

Sign
POST/auth/sign

Send mobile number of user to sign.
